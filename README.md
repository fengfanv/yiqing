# 疫情动态 `全民战疫情，贡献一份力量。`
![](/public/picture.png)
### 技术栈
1、react

2、redux

3、echarts

4、react-router

### 使用方法
1、安装依赖
```
npm install
```
2、启动项目
```
npm start
```
3、打包项目
```
npm run build
```
